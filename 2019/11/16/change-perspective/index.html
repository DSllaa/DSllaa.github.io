是时候转变视角了

发表于 2019-11-16 更新于 2021-01-28 分类于 生活 阅读次数：
本文字数： 280 阅读时长 ≈ 1 分钟
看不清是常态，但总比当楚门好。宁可看不清，我也希望自己可以拥有被遮蔽却不曾被篡改的视野。

HK的事，其实自己一直很难把自己摆到习惯里看待国际形势的鸟瞰视角。

道理很简单，我有同学在那儿。暑假的事情使我前所未有地珍视所有曾与我朝夕相处的人。生命真的太脆弱了。

不过万幸他们都回来了。只要人还是好好的，一切都可以重来。

是时候努力改回从前的视角了。就算鸟瞰中的事件仍是迷雾重重，至少鸟瞰这一行为本身便代表着力求客观理性看待问题的决心和勇气。

看不清是常态，但总比当楚门好。宁可看不清，我也希望自己可以拥有被遮蔽却不曾被篡改的视野。

中外的乌鸦一般黑，都不是什么好鸟。擦亮眼睛。既然海上没有现成的灯塔，就在自己心中建造一座任何迷雾与幻境都无法使之扭曲变形的永恒之塔。
