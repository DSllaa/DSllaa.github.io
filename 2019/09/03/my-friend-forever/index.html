西天寺一游随想

发表于 2019-09-03 更新于 2021-01-28 分类于 生活 阅读次数：
本文字数： 1.6k 阅读时长 ≈ 1 分钟
“人的一生，要死去三次。第一次当你的心跳停止，呼吸消逝，你在生物学上被宣告了死亡；第二次，当你下葬，人们穿着黑衣出席你的葬礼，他们宣告，你在这个社会上不复存在，你悄然离去；第三次死亡，是这个世界上最后一个记得你的人，把你忘记，于是，你就真正地死去。整个宇宙都将不再和你有关。”

今天第一次去学校，也算是被它的低效率震惊。回程路上，顺便去了趟西天寺。

其实也不是多么心血来潮的想法，就是很理所当然地去了。去一趟倒也不算费事，天隆寺下了地铁，直接上710坐上半个小时就到了。

进去了，问人家你的墓在哪儿，人家倒是对你印象很深。估计一方面是因为你的独特的名字，当然主要还是因为你离开时的年龄。天安园3区10排8号。你在人世间的物质残留就剩下了那冷冰冰数字所标示的一方小小的土地，想想有些不是滋味儿。

昨天我跟爸妈说了来看看你的计划，他们有些担心我一个人进坟场会害怕。可能原来也不担心，只是看了我在你出殡那天回家有些崩溃的样子，再不担心也会有些担心的。其实今天去我倒真的没啥情绪波动。坟地这种东西，无非就是个念想罢了。我不信鬼神，也没什么好怕的。其实，反倒是信鬼神的出殡时不会有我那么伤心，难道不是吗？

其实在坟场里逛一逛挺有意思的。看着有些碑上措辞的讲究与玄机，心中不禁哑然。人间的那些小心思，都渗透到亡人的世界里来了。看见有一墓碑，是夫妻的合葬墓。丈夫已然故去，也就是前两年的事，离世时才四十左右。妻子比他还小上一些，没有离去，碑上却已写上名字，只是还未上色。似乎从那位丈夫离世起，妻子便失去了再度寻找幸福的权利。不知道立碑的那位独子心中是怎么想的。在你的坟旁不愿有一座坟的主人比你大不了多少，刚到而立之年离世的。墓碑上只有女儿留了名字，也不知是习俗的原因还是其他的缘故妻子没有留名。

陵园里的环境其实很不错。绿树成荫，还有些小池塘点缀其间。当然，最主要的优点还是安静。沿着小路慢慢地走，再纷乱的心也会沉静下来。顺着之前那位阿姨的指点，我找到了你的墓。其实内心也不算很有波澜，但眼睛还是有些酸酸的。世事无常，本以为最多指的是人生的顺利或是坎坷，又怎会想到还会指人生的延续或是停止？

说到这儿，不禁想起有人的说法，觉得有些好笑。你去世后那几天，我发疯似的发了好多说说，自然就有同学来问我怎么了。我如实回答之后，有人来了一句我们还是活好当下，快乐就好。我当时就气笑出来了，可能真的是理念不同。你的离开给我的思考是，人生在世，不知什么时候就离世了，所以要努力奋斗，不留遗憾。可能相同的事情在不同的眼睛看来也是不一样的吧。

今天说是来看看你，却啥东西也没带。其实我愿意自己一个人来而不要和别人一起来也是这个原因。和别人来就不得不把考虑的重心转换到怎样才能合乎礼俗，但我知道你不会在意那些先辈们设下的条条框框。我想与清明坟前的热闹相比，你可能还是更喜欢偶尔有人单独在你坟头坐坐，来看看你。

其实正如前面所说，坟这种东西在我看来只是个念想罢了。不管人死后只是化为一抔尘土还是上天堂，去西天，或是去天蓝色的彼岸，我觉得都与我没什么关系。我真实地活在人间，想那些虚无缥缈、无根无据的东西纯属庸人自扰。在我看来，坟头这种东西就是帮助人们记得逝去的人。吴大炮曾经指出我记不清出处的一句话出自大卫·伊格曼的《生命的清单》，那句话真的是深得我心：“人的一生，要死去三次。第一次当你的心跳停止，呼吸消逝，你在生物学上被宣告了死亡；第二次，当你下葬，人们穿着黑衣出席你的葬礼，他们宣告，你在这个社会上不复存在，你悄然离去；第三次死亡，是这个世界上最后一个记得你的人，把你忘记，于是，你就真正地死去。整个宇宙都将不再和你有关。”我想我在努力做的，可能就是努力将你的第三次死亡的时间尽可能地延迟吧。

其实死亡真的是件很令人费解的东西。我之前一直在担心自己会把你忘却，因而宁愿伤心也不希望自己将你抛诸脑后。现在我自认为已经可以比较好地处理二者之间的平衡了，新的问题又出现了：我脑海中的那个你，还是你吗？会不会那个影像里混杂有太多后来的我自己的东西。

走一步算一步吧，这种东西也没有人可以给我前人的建议，至少我觉得我现在做的不错。带着对你的回忆，继承你的口头禅，好好活下去，努力奋斗吧，不留遗憾就好。

看着陵园出口呼啸而过的复兴号，我心中明白，我又回到了喧嚣的尘世，离开了那个可以短暂远离凡尘的清静之地。

有空我会常去看你的。
