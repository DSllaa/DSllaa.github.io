哪里有什么利他主义

发表于 2019-12-03 更新于 2021-01-28 分类于 生活 阅读次数：
本文字数： 531 阅读时长 ≈ 1 分钟
在我看来，天底下没有什么利他主义，一切的根源都是利己主义。

有的时候听思修老师把利己主义和利他主义作为对立的概念夸一个踩一个，心里就开始犯嘀咕。在我看来，天底下没有什么利他主义，一切的根源都是利己主义。二者表面上的差别背后只是目光长远或是短浅的区别。

我们鼓吹奉献和优先考虑他人背后的动机是什么？就鼓吹此说法的官方宣传工具来说，官方是希望通过道德来部分代替法律，通过鼓吹奉献的精神营造鼓励此类行为的社会风气，从而达到用舆论的力量部分代替建构成本高昂维护成本不菲的国家机器。对于我们平常人，这种跟风鼓吹的背后有的只是身在人群中盲目性的从众和对于被孤立的恐惧，以及奉献行为之后的自我感动。

人们总是否定利己主义，并且试图回避自己的自私心理，在我看来大可不必。与其假惺惺地自称自己的利他行为是为了别人，不如老实地承认是为了自己——为了自己被别人褒奖，为了自己的好人缘，为了不被他人孤立或是戳脊梁骨，为了帮助他人之后获得心理上的崇高感和自我感动。

直面自己身为人的自私的特点，不必遮遮掩掩。按照达尔文的理论，从心里根源上不自私的人理应早已被历史淘汰，因而自私被深深地镌刻进人类的基因并不是那么难以理解。

同时，也要在宣传工具火力全开的时候保持自己的判断。否则就只是做棋子的命，就不能掌控自己的命运。你哪里知道下棋的人什么时候要拿你去对子？
